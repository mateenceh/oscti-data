2019 Gartner Magic Quadrant for Network Firewalls

Forcepoint recognized as a Visionary for 3rd year in a row in the Gartner 2019 Magic Quadrant for Network Firewalls

Gartner has named Forcepoint a Visionary in their 2019 Magic Quadrant for Network Firewalls for ability to execute and completeness of vision.

According to Gartner, “With firewall providers embedding multiple security features in firewalls and enabling integration and automation capabilities with other security products, firewalls are evolving into network security platforms.”*

Forcepoint believes this evolution is best achieved by combining three areas:

  • Enterprise Connectivity – Direct-to-cloud application routing combined with site-to-site VPNs, made scalable and resilient with advanced, mixed-device clustering.
  • Extensible Security – Market-leading intrusion prevention and access control that’s integrated with cloud-based Forcepoint Web Security and Cloud Application Security Broker (CASB).
  • Management at Scale – Real-world experience handling more than 1800 sites from a single pane of glass with zero-touch deployment and rapid, interactive incident response.

Download the Gartner report to learn more.

 

*Gartner, Magic Quadrant for Network Firewalls, Rajpreet Kaur, Adam Hils, Jeremy D'Hoinne, John Watts, 17 September 2019
